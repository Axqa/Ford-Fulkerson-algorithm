Control:

  • Double-click on empty space: Add a new node
  • Left-click on a node: Select the node
  • Drag from border to empty space: Create new node with edge
  • Drag from border to node: Connect nodes with edge
  • Drag node (not from border): Move node
  • Drag head of edge: Move edge
  • Double-click on a node: Edit text
  • Double-click on an edge: Edit edge max value
  • Press 'Delete': Delete selected node or edge
  • Drag green label on selected node to adjust label position
  • Drag text on edge to move it

Information:

Path between starting and target nodes existence will be displayed below the graph.

If path exists, algorithm steps would be displayed below.
